# 결정 로그 (Decisions Log)

두 에이전트(Claude Code, Codex CLI)가 합의했거나, 사용자가 최종 판단한 설계/방향 결정을 기록.

형식:
```
## [YYYY-MM-DD] 제목
- **결정**: <한 줄 요약>
- **이유**: <왜 이렇게 정했는지>
- **대안**: <고려했지만 채택하지 않은 옵션>
- **참여**: claude / codex / user
```
